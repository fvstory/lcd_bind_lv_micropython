:mod:`lcd` --- LCD Driver library
==================================

.. module:: lcd
    :synopsis: LCD Driver library

该模块包含市面上常见的 LCD 驱动，如 SPI LCD、I80 LCD（又名 Intel 8080并行LCD）、RGB LCD等。

目前SPI LCD还在准备中。
